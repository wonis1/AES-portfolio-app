# Soccer Performance Tracker

## 1. 프로젝트 소개

아마추어 축구 선수의 경기 데이터를 자동으로 수집하고,
AI 기반으로 퍼포먼스를 분석하는 웹 애플리케이션입니다.

사용자는 별도의 수동 입력 없이,
스마트워치 데이터를 기반으로 경기 기록과 피드백을 받을 수 있습니다.

---

## 2. 내가 해결하려 한 문제

아마추어 선수는 자신의 경기 퍼포먼스를 객관적으로 기록하기 어렵습니다.

- 거리, 속도, 활동량은 기억에 의존
- 피드백은 감정 기반 평가
- 개선 방향이 정량화되지 않음

이 문제를 자동 수집 + AI 분석 구조로 해결하고자 했습니다.

---

## 3. 핵심 기능

### 3-1. 자동 데이터 수집
- Google Fit API 연동
- 경기 시작 시 자동 세션 생성
- GPS 기반 이동 거리 계산

### 3-2. AI 분석 모듈
- Python 기반 분석 서버 분리
- Pandas로 활동량 계산
- 경기별 퍼포먼스 등급 산출

### 3-3. 대시보드
- React + Vite 기반 SPA
- 경기별 기록 시각화
- 성장 추이 그래프 제공

---

## 4. 기술 스택 선택 이유

- React: 상태 기반 UI 관리
- Supabase: 빠른 인증/DB 구축
- Python: 데이터 분석에 최적화
- Vercel: CI/CD 자동화

---

## 5. 기술적으로 어려웠던 점

- Google Fit 데이터 구조 파악

- 자동 세션 생성 로직 설계
