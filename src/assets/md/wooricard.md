# 📌 우리카드 챗봇 재구축 프로젝트

**기간**: 2023.08 ~ 2024.04 (9개월)  
**역할**: 프론트엔드 개발

---

## 🧩 프로젝트 성격

금융권 챗봇 서비스 재구축 프로젝트  
시나리오 로직 설계, 인증 구조 정리, 상담 솔루션 연동, 보안 요구사항 반영을 수행

---

## ⚙ 주요 수행 내용

### 1️⃣ 세션 기반 인증 컴포넌트 구현
- 로그인/비로그인 사용자 분기 로직 구현
- 인증 상태 기반 조건 처리 구조 개선
- 시나리오 내 인증 흐름 일관성 확보

---

### 2️⃣ 상담 솔루션 연동 및 상태 유지 흐름 구현

- 챗봇 → 상담 솔루션 전환 흐름 정의
- 상담 솔루션 → 챗봇 Open API 연계 구조 협의
- DB 데이터 기반 코드 전달 구조 반영
- 상담사 버튼 클릭 시 세션 유지 상태로 챗봇 복귀
- 시나리오 종료 후 기존 상담 히스토리 유지 흐름 구성
- 타임아웃 및 예외 흐름 정리
- Kubernetes 파드 접속 및 gz 로그 분석을 통한 장애 원인 추적

협업 대상:
- 우리카드 내부 개발자
- 상담 솔루션 개발자
- 상담 현업 담당자

---

### 3️⃣ 금융 보안 및 처리계 연동

- 보안 앱 설치 여부 체크 로직 구현
- Java 기반 처리계 연동 과정 이해 및 적용
- 프론트 → 처리계 → 보안 시스템 흐름 경험

---

### 4️⃣ 웹 접근성 대응

- ARIA 속성 수정
- tabindex 속성 추가
- 키보드 네비게이션 개선
- 포커스 이동 로직 보완
- 접근성 심사 기준 반영

---

## 📎 사용 기술

JavaScript  
HTML / CSS  

챗봇 솔루션 기반 개발  
상담 솔루션 Open API 연동  
세션/상태 관리  

Kubernetes 운영 환경 로그 분석  
Java 처리계 연동 경험
